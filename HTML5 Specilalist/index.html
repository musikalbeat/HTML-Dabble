Hyperlinks

This is section, we will be looking at different types of links. Links are found in nearly all web pages. Links allow users to click their way from page to page. Note that a link does not have to be text. A link can be an image or any other HTML element.

Anchors to Navigate Long Pages

These are called Named Anchors or Bookmarks. This helps us navigate long pages to get to the information we need faster.

We will use an empty anchor tag with only the name attribute filled. <a name=""></a>

Playing and Controlling Audio

To play an audio file, we need to use the audio tag. Inbetween the opening and closing audio tag, we put the source tag and then the src element. In addition, if we want to be able to control the playback and volume of the audio, we include the controls attribute with the value "controls". You can also include the autoplay attribute too!

Sometimes a browser would not accept a type of audio file, such as mp3. We can include another source audio file if the one prior does not play.

Display Data in Tables and iFrames

In the early days of the web, tables were used to layout pages. This method isn't used anymore as we now use CSS (Cascading Style Sheets) for page layout.

Defining a Table

Previously, we know that people do not like to read long passages of texts. A technique we can use to section off the information is by using a table.

In order to create table, we use the <table></table> tags. Inbetween those tags goes the data for the table. Each row of the table uses the <tr></tr> (table row) tags. And then we can define each data using the <td></td> (table data) tags.

Defining Border of a Table

For this topic, we will learn how to turn table borders on and off.

As the table gets longer and longer, it will become harder to read. To turn on the borders, add the border attribute to the table tag and assign it a numerical value. To give space inbetween the border lines, use the cellspacing attribute. To give the data inside each table cell space between the border and the data, use the cellpadding attribute.

Name Age

Due note that is method is an older method. The more contemporary method is configuring its attribute in a style sheet.

Determining Table and Cell Width

If you are concerned by table appearances on mobile devices, then this topic will show you how to handle it. This topic will show you how to make the table width more responsive.

Much like how we had images span to the sizes of different window lengths, we will use the same width attribute to have the table be responsive by assigning a percentage value. For a fixed size window, assign a pixel unit value.

You can also change the width of a table column by using the same width attribute in a table data tag. Using a percentage value will have it be a fraction of the whole table width (not window page).
