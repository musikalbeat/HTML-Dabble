Lesson 9: Building a CSS Telephone

Universal Selector

The universal selector, otherwise known as the asterisk (*) symbol, selects all elements in the document.

The syntax is as follows:

* {
property: value;
}

The following code would change all text color in a given document to red.

* {
color: red;
}

Attribute Selector

The attribute selector, selects each element that matches a specific attribute. To recall, an attribute could be something like an href or a type on an HTML element.

Using a selector with an additional attribute included in square brackets, we can target all elements that match that selector and attribute.

selector[attribute] {
property: value;
}

For example, if we wanted to select all elements that include an href attribute and add padding, we can use the following code:

a[href] {
padding: 50px;
}

Not only can we simply select elements that match a selector and attribute, but we can select all elements that match a selector and a specific attribute value.

selector[attribute = "some value"] {
property: value;
}

For example, if we wanted to select all elements that include a type attibute that matches the value of "submit" and add margin, we can use the following code:

button[type = "submit"] {
padding: 50px;
}

Descendent Combinator

The descendent combinator, selects all child elements of a specific selector that descend (e.g. children, grandchildren, great grandchildren, ...) from a selector. The type of element does not need to be direct descendent from the parent )or ancestor) selector (Direct Child Combinator).

To use selector, all we need to do is use a space between the selector and its descendent's selector.

selector selector {
property: value;
}

If we wanted to select all p tags elements that are a children of a div tag and set their font-size, we use the following code:

div p {
font-size: 12px;
}

However, we can also use classes and ids. If we wanted to select all img tags elements that descend from an id of "main" to set their width, we can use the following code:

#main img {
width: 500px;
}

Direct Child Combinator

The direct child combinator, selects all child elements of a specific selector that are direct children from a parent selector. The direct child combinator is nearly identical to the descendent combinator, except it does not allow selection of grandchild, great grandchildren, and so on. It only targets direct children.

To use the direct child combinator, we use a right chevron, or greater-than sign ( > ).

selector > selector {
property: value;
}

parent-selector > child-selector {
property: value;
}

#header > .title {
color: blue;
}
