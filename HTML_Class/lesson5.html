Units

We have already seen how to declare a size in units for things like height/width, margin/padding, font-size, etc. However, we've only been using pixels )px) and percentages up to this point.

rems - relative units that refer to the font-size of the root element. (ex. 5rem)

ems - relative units that refer to the font-size of the current element. (ex. 5em)

viewport height (vh)/viewport width (vw) - viewport is just a fancy way of saying "the browser window", so referring to a value such as "50vw" would refer to 50% of the viewport width and "20vh" would refer to 20% of the viewport height.

The value of the viewport height and width actively changes as the user adjusts the size of the viewprt by draggin the window larger or smaller.

Overflow

Overflow is a CSS property that changes what happens to content when it is too big to fit into an area.

Over has four notables values: visible, hidden, scroll, and overflow-y.

Opacity

Opacity defines the transparency of an element on a scale of 0.0 to 1.0

Transparency can be useful in a number of ways when styling in CSS, whether to make an element less visible or for stacking elements on top of each other using relative or absolute positioning.
