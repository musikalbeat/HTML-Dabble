What are Pseudo-classes?

A pseudo-class is used to define a special state of an element. For example, we'll look at the hover, active, and focus pseudo-classes in the lesson which each represent a certain state. For example, hover pseudo-class will be triggered when the user hovers their mouse over an element.

If we think about it this way, a pseudo-class is just another class applied to an element when a specific event occurs.

The pseudo-class syntax is as follows:

selector:pseudo-class {
property:value;
}
