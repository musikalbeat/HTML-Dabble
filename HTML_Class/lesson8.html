Our First Flexbox

What is Flexbox?

Flexbox is a 2D grid used for laying out elements in CSS. We can use flexbox whenever we want to lay elements out vertically or horizontally. It is significantly better for displaying elements in a grid then using margins, because with flexbox, we are able to have our elements laid out for us without having to do much work.

How do we create a Flexbox?

To create a CSS flexbox, all we need to do is set the display of our parent container to flex.

.parent-container {
display: flex;
}

flex-direction

We can easily change the direction of our flexbox using the flex-direction property. Be default, all flexboxes have their flex-direction set to row. This means that they will be displayed from left to right (horizontaly). However, we can easily change our flexboxes to be displayed up and down (vertically) when floex-direction is set to column.

We place the flex-direction property on the parent.

flex-wrap

When a flexbox becomes too tall or too wide to be shown on one screen, the overflowing elements will be shown by default. However, if we wanted to wrap our flexbox when it is too long, we could flex-wrap property to wrap.
