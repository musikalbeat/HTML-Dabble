Lesson 4: Laying Out Element

The CSS position sets how an element is positioned on a page. There are many options for ways an element can be positioned, however we will only be looking at: static, relative, and absolute positioning.

Static Positioning

Static positioning is the default way that elements are positioned on a page. Thay follow the standard box model rules and interact naturally with the flow of the page.

This is static.

Relative Positioning

Relative positioning causes an element to positioned relative to its parent element.

For example, setting a relative element's height and width to 100% will match its parent element's height and width.

Elements with relative position, similar to static position, interact naturally with the flow of the page.

Parent
Child
position: relative
top:50%;
left:50;

Absolute Positioning

Absolute positioning causes an element to be positioned in relation to its most recently position ancestor. If there are no positioned ancestors of an element that is absolutely positioned, it gets positioned in relation to the "body" element.

A major difference about elements that are absolutely positioned is that they are removed from the normal flow of the page. This means that they can lay on top of other elements flawlessly, and do not follow box model rules with its surrounding elements.

Parent
position: relative;
Child
position: absolute;
top: 50%
left:50%;

Types of Display

There are many types of display in CSS, but we will only focus on three for now. There is inline, block, and inline-block.

Inline elements cause height and eidth properties to have no effect. They are allowed to be on the same line os other inline elements (ex: button).

Block elements start on a new line and take up a full line. They cannot sit on the same line as inline elements (ex: p).

Inline-block elements are formatted as an inline element, but you can also apply height and width values, unlike inline element. No elements are inline-block display by default, but all elements can be set to inline-block.
