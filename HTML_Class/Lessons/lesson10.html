What is Responsive Design?

Responsive design is the principle that all applications should be optimized to fit on every screen size. A responsive web page will automatically adjust for different screen sizes and viewports.

Below is a brief look at the Apple's website to see their responsive design in action.

Media Queries

We can create our own responsive design in CSS using media queries. Media queries are conditional declarations of styling that only apply based on a given condition about the viewport (think of them almost like "if statements" in other languages).

We can nest multiple new selectors and declarations within w media query. The following code shows how to do so, using our media query syntax of "@media".

@media (some condition) {
selector {
property: value;
}
selector {
property: value;
}
selector {
property: value;
}
}

Min-Width & Max-Width

Two conditionals within the viewport are the min-width and max-width. Not only are these properties able to be set in general to CSS, but they determine how big or small an element is allowed to become. But we can use them determine the screen size within a media query declaration.

Min-width determines whether a screen is greater than a given size. For example, we could use a media query that uses min-width to check if the screen size is greater than 800px in width.

@media (min-width: 800px) {
p {
font-size: 12px;
}

Max-width determines whether a screen size is smaller than a given size. For example, we could use a media query that uses max-width to check if the screen size is smaller than 800px in width.

@media (max-width: 800px) {
p {
font-size: 8px;
}
