What are Transformations?

The transform property applies a transformation to an element. This property allows you to translate, rotate, scale, skew, etc. However we will only be looking at translate and scale, as they are the most easily applicable transformations in CSS.

Transformation syntax is as follows:

div {
transform: transformationType(value);
}

Translations

The translate property allows us to move an element in any 2D direction.

However, to understand translations in CSS, we must understand how the X and Y axis are set up. Unlike a normal four quadrant plane in algebra, CSS flips and Y axis so that moving down is a positive Y value.

For example, if we wanted to move a rectangle down and right, we'd use a positive X and Y value in the translation.

We can use translations for the X and Y axis specifically, using translateX and translateY.

div {
transform: translateX(50px);
}

div {
transform: translateY(20px);
}

However, we can use translate while providing the Xand Y value to control both at the same time.

div {
transform: translate(50px, 20px);
}

Scale

The scale transformation allows us to multiply the size of an element by a specific value. Elements are at a scale value of 1 by default, referring to 100%. Scale an object to 2 would double its size, reffering to 200%.

We can use decimals to scale an object down. For example, this would scale a div to 50% of its original size.

div {
transform: scale(0.5);
}

Combining Transformations

By default in CSS, writing a property multiple times in a declaration block sets us up to fail. CSS will always apply the last property, rather than the duplicate ones before it.

For example, the background color will be set to blue below:

div {
background-color: red;
background-color: green;
background-color: blue;
}

When it comes to transformation, this is no exception. Declaring multiple transformation properties will only apply the last one declared. However, we can combine the multiple different transform values into one line by separating them by spaces.

Incorrect Code:

div {
transform: translateX(50px);
transform: translateY(20px);
transform: scale(0.8);
}

Correct Code:

div {
transform: translate(50px, 20px) scale(0.8);
}
